# eFootball Tournament System - User Guidelines

## Overview
This platform lets you wager KES on eFootball matches against friends. The system accepts **only eFootball-generated room codes** - we do NOT generate our own codes.

---

## How to Create a Match

### Step 1: Create a Room in eFootball
1. Open **eFootball** on your console/PC
2. Go to **Friends Match** (offline mode to invite a friend)
3. Set up the match with your desired teams
4. When the room is created, **eFootball will generate a room code** (e.g., `ABC123` or `XYZAB`)
5. **Copy this code**

### Step 2: Enter the Code on Our Platform
1. Open this platform and go to **Create Match**
2. Enter your **wager amount** (minimum KES 50)
3. Paste the **eFootball room code** from step 1
4. Confirm - your wager amount will be deducted from your wallet
5. Share the code with your friend

### Step 3: Your Friend Joins
1. Your friend opens this platform → **Join Match**
2. They enter the **SAME eFootball room code**
3. They also need to wager the same amount (this will be deducted from their wallet)
4. Match is now **ACTIVE**

---

## How to Play & Settle

### Playing the Match
1. Both players play the match in eFootball using the room code
2. Play until the match ends
3. Note the **final score** (e.g., 3-2)

### Uploading Results
1. Take a **screenshot of the final result screen** in eFootball
2. Come back to this platform
3. Go to your active match and upload the screenshot
4. The system will use **OCR** to read the score and detect team names automatically

### What Happens Next?
- **85%+ confidence**: Match auto-settles immediately, winner gets paid
- **50%-85% confidence**: Opponent has 2 hours to challenge
- **<50% confidence**: Manual approach needed (manual score declaration)

### If There's a Draw
- If the match ends in a draw, you'll be asked to play a **Penalty Shootout** in eFootball
- Create a new room code for the penalty shootout
- Upload the penalty result screenshot to determine the winner

## Important Notes

### Code Format
- eFootball codes are typically **4-8 alphanumeric characters**
- Examples: `ABC123`, `XYZAB`, `12AB`
- Country code format DOES NOT work - use only the room code

### Supported Teams
Our system recognizes 100+ eFootball teams including:
- Premier League: Arsenal, Liverpool, Manchester City, etc.
- La Liga: Real Madrid, Barcelona, Atlético Madrid, etc.
- Serie A: Juventus, Inter Milan, AC Milan, etc.
- Bundesliga: Bayern Munich, Borussia Dortmund, etc.
- Ligue 1: PSG, Marseille, Lyon, etc.
- International teams and more

If your team isn't recognized, you can still play but may need to manually confirm the result.

### Team Names in Your Profile
Make sure you set your **team name in your profile** (not just username). This helps with automatic winner detection.

## Troubleshooting

### "Invalid eFootball code format"
- Check the code you entered - it should be 4-8 characters
- Copy directly from the eFootball room creation screen
- Don't add dashes or extra characters

### "This code is already in use"
- Someone else is already using this code
- Create a new room in eFootball and get a new code

### "Match is not active"
- Ensure your friend has already joined the match
- The match expires after 30 minutes if unclaimed

### "Screenshot confidence too low"
- Make sure the final score is clearly visible
- Try uploading the screenshot from the end-of-match summary screen
- Avoid partial screenshots or cropped images

### "Could not identify your team"
- Take a screenshot that shows both team names on the result screen
- Make sure your team name in the profile matches the team in eFootball

## Settlement Methods

Your match can be settled in several ways:

1. **Auto-Settlement (85%+ confidence)**: OC reads the score and determines winner automatically
2. **Challenge Window (50-85% confidence)**: Opponent can upload contradicting evidence within 2 hours
3. **Manual Declaration**: Player declares the score, opponent has 30 minutes to confirm or dispute
4. **Penalty Shootout**: For drawn matches
5. **Admin Review**: If fraud flags are raised or scores are disputed

## Safety & Fraud Prevention

The system checks for:
- ✅ Duplicate/recycled screenshots
- ✅ Edited/manipulated images
- ✅ Screenshots taken before the match started
- ✅ Suspicious win rates
- ✅ Rapid submissions
- ✅ Team name matching
- ✅ Score sanity checks

If any red flags are raised, your match goes for **admin review** before payout.

## FAQ

**Q: Can I use my own code instead of eFootball's?**
A: No. The system only accepts eFootball-generated room codes to prevent duplicate codes and collisions.

**Q: What if eFootball crashes during the match?**
A: You'll need to create a new room and get a new code. The previous match will expire after 30 minutes if unclaimed.

**Q: Can I cancel a match after joining?**
A: Once both players have joined and the match is ACTIVE, you can forfeit which counts as a loss. Pending matches (before joiner joins) can have wagers refunded.

**Q: What's the maximum wager?**
A: There's no technical limit, but withdrawals are capped at KES 100,000 per day.

**Q: How long does settlement take?**
A: Auto-settlement is instant. Challenge windows last 2 hours. Admin review typically takes < 24 hours.
